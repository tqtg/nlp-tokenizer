## Usage :

Run this command in lib folder:
 - For Linux: <br/>
 `java -classpath args4j-2.0.6.jar:jmdn-base.jar:tokenizer.jar main.Main -filename ../data/chudu24.txt`

 - For Windows: <br/>
 `java -classpath args4j-2.0.6.jar;jmdn-base.jar;tokenizer.jar main.Main -filename ..\data\chudu24.txt`

**NOTE:** This project is built in Java 1.7
